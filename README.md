## About This Project

This is my personal portfolio website, designed and built from scratch to showcase my work as a Data Scientist, AI practitioner, and writer. It features my projects, professional experience, certifications, Medium articles, and a contact form — all in a clean, dark-themed, fully responsive design.

---

## Features

- **Animated role typewriter** — cycles through job titles with correct grammar (`a` / `an`)
- **Dark / Light / System theme toggle** — respects OS preference and saves to localStorage
- **Filterable project cards** — filter by AI/ML, Data Analytics, Power BI, or Statistics
- **Vertical experience timeline** — clean alternating layout
- **Medium blog section** — live article cards linking to published posts
- **Certifications section** — education and professional credentials
- **Fully responsive** — works on mobile, tablet, and desktop
- **No frameworks** — pure HTML, CSS, and vanilla JavaScript

## Getting Started Locally

### Prerequisites
- [Node.js](https://nodejs.org/) (v16 or higher)
- [Git](https://git-scm.com/)

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/Omaa-analyst/mmesoma-portfolio.git

# 2. Navigate into the folder
cd mmesoma-portfolio

# 3. Install dependencies
npm install

# 4. Start the dev server
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

### Build for Production

```bash
npm run build
```

Output goes to the `dist/` folder — ready to deploy anywhere.
